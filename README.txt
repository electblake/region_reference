-- SUMMARY --

Provides CCK Field for referencing regions (of active theme)

-- REQUIREMENTS --

* CCK 2.x+


-- INSTALLATION --

* Enable module in admin/build/modules


-- CONFIGURATION --

* Add field as you would normally and choose Region Reference
